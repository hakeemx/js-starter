#React hot reload starter pack

This is a starter pack I made to help you get started and speed up your developer workflow. This includes:
React,Redux,Webpack,Babel and Broswersync for hot reloading.

#Installation
If you have yarn installed run "yarn" and this will install all the required dependencies. If not then you can install run npm install.

#Getting Started
To get started simply run "npm run dev" to start the server.
